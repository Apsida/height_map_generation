Название: Библиотека по генерации карт
Проект представляет собой консольное приложение на языке C, которое автоматическисоздаёт карты высот — изображения,
отражающие рельеф местности (например, горы,равнины, впадины). Затем оно размещает объекты на этой карте (подземелья,
замки, города). Алгоритм Diamond-Squareприменяется для генерации естественно выглядящего ландшафта, имитируяслучайные,
но плавные изменения высоты. Пользователь указывает размер карты ижелаемую «шероховатость» рельефа.

Описание:
struct Biome - структура хранящая параметры биома: Названия, границы в которых появляется.
struct Obj - структура хранящая параметры объекта: название, биом в котором появляется, количество на карте.
struct Tile - структура хранящая параметры клетки (или тайла) карты: Высота, температура, указатели на биом и объект.
Tile_param - хранит список параметров по которым можно обращатся в некоторых функциях

map_IO - здесь представлены основные функции для задания параметров карты и её заполения:
	Tile** allocate_map(int size) - выделяет память под карту
	void set_seed(unsigned int seed) - выставляет сид(если передать 0, то сид будет случайный)
	void print_map(Tile** map, int size, tile_param param); - выводит в консоль карту по определённому параметру
	void fill_map(Tile** map, float** filler, tile_param param, int size, int max_val); - используя сгенерированную
	карту коэффициентов заполняет соответстующий параметр в карте
	void save_info(Tile** map, int size); - сохраняет основные параметрыы карты (высота и температура) в файлы

generation_alg - здесь создаёться двумерный массив коэффициентов (float) по алгоритму diamond-square:
	float** diam_sq_alg(int size, float roughness) - функция принимает параметры размера и шароховатости и выдаёт
	двумерный массив коэффициентов(от 0 до 1)

С помощью функции diam_sq_alg() создаём массив коэффициентов и передаём его в filler_map с параметром максимального 
значения для переменной. Таким образом заполняем map картами высот и температур.

После инициализации карты map нужно проинициализировать списки типов объектов (Obj* obj_list) и биомов (Biome* biome_list).
В данной программе есть функции для их инициализации путём задания через консоль. 

biome_init_alg - здесь представлен алгоритм по заполнению карты биомами:
	void biome_init(Tile** map, Biome* biome_list, int size, int biome_count, Cmp_Func cmpf) - принимает ссылки на
	карты и лист типов биомов, количество типов биомов и функцию согласно, которой будет производиться определение
	какой биом присваивать клетке. Функция сравнения должна использовать параметры ограничения по высоте и температуре.

objs_place_alg - здесь представлен алгоритм по расставлению на карте объектов:
	void objs_place_alg(Tile** map, int size, Obj* obj_type, int obj_type_count) - функция принимает ссылки на карту
	и список типов объектов. Для каждого типа объекта итерационно случайным образом выбираются точки и проверется
	совпадение название биомов

str_lib - здесь находятся вспомогательные функции для работы со строками:
	char* str_read(); - считывание строки из консоли
	int str_cmp(char* a, char* b); - сравнение строк
	int biome_check(Tile** map, int size, char* biome); - подсчёт количества биомов с названием из параметра biome

UI - user interface сделанные для получения данных из консоли:
	Tile** map_init(int* size, int* max_height, int* max_temper, unsigned int* seed, double* roughness); - функция
	которой надо передать указатели на основные параметры (эти параметры проинициализируйте заранее).
	Biome* receive_biome_list(int* biome_count, int max_h, int max_t); - функция принимает указатель на заранее
	принициализированную переменную количества типов биомов, а также ограничения для вводимых лимитов для соответствующих
	параметров
	Obj* receive_obj_list(Tile** map, int size, int* obj_count); - функция принимает указатель на заранее 
	принициализированную переменную количества типов объектов, а также указатель на карту, для того, чтобы проверять 
	корректность вводимых данных (например существует ли биом указанный для объекта в параметрах карты и сгенерировалься
	ли он при текущих основных параметров генерации)
	void view_count_biomes(Tile** map, int size, Biome* biome_list, int biome_count); - показывает количество всех 
	инициализированных биомов.
